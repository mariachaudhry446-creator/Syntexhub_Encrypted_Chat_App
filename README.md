## Encrypted Chat App

A secure client/server chat application with AES encryption.

## Features
- ✅ AES-256 encryption using Fernet
- ✅ TCP socket communication
- ✅ Multiple clients support (threading)
- ✅ Real-time encrypted messaging
- ✅ Secure key handling

## How to Run

### Install dependency:
      pip install cryptography
      
## Start Server:
    python server.py
## Start Clients (in new terminals):
    python client.py
## Default Commands:
   Type message and press Enter to send
  Type /quit to exit chat
